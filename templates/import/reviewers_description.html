This import script allows you to import your pool of reviewers from a CSV file with the following structure:

Salutation, firstname, middlename, lastname, email_address, department, institution, country, reviewer interests

If the user is not in the system yet, you can toggle the flag below to start the review process automatically for all users.

Even when using the automatic password reset process, it is important to let your reviewers know ahead of time.

Here is an example line from a dummy csv:

Prof, Martin,Paul,Eve,mpeve@voyager.com,Science,USS Voyager,UK,'Humanities;Open Access'
